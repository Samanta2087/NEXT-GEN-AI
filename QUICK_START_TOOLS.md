## Get Started in 3 Minutes

### 1. Navigate to Tools
Visit `/tools` in your browser after starting the dev server.

### 2. Choose Your Tool
Pick from 15 professional tools:

5. Download individual PDFs

## üí° Pro Tips

### Batch Processing
- Upload up to 50 files at once
- Select multiple files in left panel
- Process all with same settings
- Download as ZIP

### Before/After Comparison
- Select a file to see preview
- Use the interactive slider
- Compare original vs processed
- Toggle split view on/off

### Keyboard Shortcuts
- `Ctrl/Cmd + V` - Paste image from clipboard
- `Ctrl/Cmd + S` - Quick save/process
- `Escape` - Back to tool selection
- `Space` - Toggle preview mode

### Quality Guidelines
- **Web**: 70-80% quality, WEBP format
- **Print**: 90-95% quality, PNG format
- **Archive**: 100% quality, PNG format
- **Thumbnails**: 60-70% quality, JPG format

1. **Use WEBP** - 30% smaller than JPG, modern browsers
2. **Target Size Mode** - Auto-finds optimal quality
3. **Batch Similar Files** - Process same dimensions together
4. **Disable Auto Optimize** - For manual control
5. **Lock Aspect Ratio** - Prevent distortion

## üêõ Troubleshooting

### Upload Failed
- Check file size (max 50MB per image)
- Verify format (JPG, PNG, WEBP, SVG, HEIC)
- Try refreshing the page

### Processing Stuck
- Large files take longer
- Check browser console for errors
- Try processing fewer files at once

### Preview Not Showing
- Enable split view toggle
- Select a file in left panel
- Refresh page if needed
